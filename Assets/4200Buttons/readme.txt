4200 Common Buttons Pack 

This is a mobile-friendly buttons collection which contains more than 4200 buttons 
in 21 pre-defined themes, 
and 100 separated icons for easily customizing.

All assets are graphics PNG files have 128x128 pixels resolution. 

It also contains the high quality PSD files so you can create new assets based on them!


Features:

- 100 icons; 

- 21 themes include 7 color: red, orange, green, blue, purple, yellow, gray;

- 204 buttons for each theme :
  101 round buttons, 101 square buttons, 1 rectangle button, 1 oval button;

- 4 shapes of button background: round, square, rectangle, oval.

- 1 ui background.
